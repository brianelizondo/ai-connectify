Class: HttpClient

HttpClient(baseURL, defaultHeadersopt, timeoutopt)

Represents an HTTP client for making API requests using Axios. This client is designed to make HTTP requests to a specified base URL and includes error handling specific to AI-Connectify.

new HttpClient(baseURL, defaultHeadersopt, timeoutopt)

Create a new HttpClient instance. Initializes the Axios client with a base URL and optional default headers.
