(inner) createAIInstance(ai, apiKeyopt) → {object}

Creates an instance of the AI connector for the specified AI This function retrieves the AI class and API key requirement from the AIRegistry, validates the API key if needed, and returns a new instance of the AI class.
Parameters:
Name Type Attributes Default Description
ai string The name of the AI to use (e.g. TensorFlowNode, ChatGPT, DALLE, or Cohere)
apiKey string <optional>
 null The API key required for some AIs. It can be null if the AI does not need an API key
